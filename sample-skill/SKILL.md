# SEO Writer

## Description
Generate SEO-optimized blog posts with keyword analysis.

## Instructions
1. Ask the user for a target keyword and topic
2. Research LSI (Latent Semantic Indexing) keywords
3. Create an outline with H2/H3 headings
4. Write a 700-word blog post following SEO best practices:
   - Keyword in title, first paragraph, and conclusion
   - Use 3-5 LSI keywords naturally throughout
   - Include meta description (155 chars max)
   - Add internal linking suggestions
5. Output the formatted markdown blog post

## Example Usage
```
User: Write an SEO blog post about "best coffee grinders 2026"
```

## Compatibility
- Claude (via .claude directory)
- Cursor (via .cursor directory)
- GitHub Copilot
